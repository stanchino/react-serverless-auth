## Installation

```bash
$ npm install --save react-serverless-auth

or using yarn:

$ yarn add react-serverless-auth
```

## Demo

```bash
$ npm run example

or using yarn:

$ yarn example
```

## Development

```bash
$ npm test

alternatively

$ yarn test
```
